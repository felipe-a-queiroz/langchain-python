# Aprendizado LangChain com Python

Este repositório é dedicado ao aprendizado de LangChain com Python. O roteiro de aprendizagem completo é o seguinte:

## Conectando na OpenAI
- Conectando com API da OpenAI via LangChain
- Otimizando a gestão de consultas médicas com LangChain
- Utilizando um prompt template com LangChain

## Criando cadeias
- Criando cadeias com LCEL no LangChain
- Definindo a resposta do modelo como um objeto JSON
- Criando uma sequência de cadeias com LCEL

## Interações com o chat
- Simulando uma interação de chat sem memória
- Adaptando o chat para utilizar LCEL
- Adicionando memória ao chat com LangChain

## Orquestração
- Criando a base de uma solução com LCEL e LangGraph
- Orquestrando assistentes sem LangGraph
- Orquestrando assistentes com LangGraph

## Implementando RAG
- Usando RAG em um txt com LangChain
- Usando RAG em vários PDFs com LangChain
